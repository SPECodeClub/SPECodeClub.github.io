To start off with, we need to set up our html file, and learn how to open it on a computer. First, open notepad by pressing the windows button and typing 'notepad', then pushing enter.



Now, we need to type out the following code:

<!DOCTYPE html>
<html>
<body>


</body>
</html>




Then, we have to save the file as a .html file. To do this, select 'File' and 'Save as', then select the menu under where you type the filename and select 'All files'. Then, type your file's name followed by .html




Code breakdown:
<!DOCTYPE html> - tells the computer it's a html file
<html> - code goes inside this tag!
<body> - body of the code goes in here


</body> - ends the body of the code
</html> - ends the html code


Now, we can go into the folder where we saved our file, right-click on it and select 'Open with - Google chrome'. Ta-da! You have created your first website. It looks pretty blank right now so turn over to the next page to make some content!

To add some content to our page, we need to be aware of four tags.
The first is <title> and </title>, which is the title for our page and will appear in the tab at the top. When people have multiple tabs open at once on Google, they need to know which one is which!



Next, we have <h1> and </h1>, which are heading tags. Anything within them will become a header. The lower the number, the larger the header - it's between 1 and 6.



Try reloading your website. If you've added those tags, it should now have a title and a heading! Well done!
After this, we'll need some proper text. It's not a very good website without a bit of content! The tags for this are <p> and </p>. Try it out!



Finally, we need to learn about line breaks. Currently, if you do something like this, the lines of text will just appear next to each other:

<!DOCTYPE html>
<html>
<title>My website</title>
<body>


<h1>My First Heading</h1>
<p>
This is the text in my website!
I like my website.
</p>


</body>
</html>

To fix this, we add line breaks. The syntax (code) for them is <br>. You can add one for a new line, or two for a new paragraph!



Try your website out. Wow! It's really coming together nicely. Have a look at the next page for a few more features!

Page Three

Our website has text, but its a bit boring on its own. We need a way to make our website more advanced - images!

To add images, we need to have an image. We can use another image on the web, but it's better to download one. Right click on your image, then select 'Save as'. Save it in the same folder as your .html file! If you are using one online without downloading it, right click on the image and choose 'Copy link address'. Make sure you are using a free image - we don't want copyright issues.
To add the image, write the following tag into your code:

<img src="[change this to your file's location]" alt="[The alternative text]">

Your file's location needs to be the name of your image. If you are using one from the web, paste the url you copied earlier there. The alt text is used if somebody cannot see the image (for instance if their browser is too old to show images, or they are blind and use narrator to interact with the computer). The alt text is shown next to a small icon of an image (indicating the image cannot be shown), or read aloud in these cases.



Try it out. Ta-da! Your website is now interesting!
Next, we need to spice things up a little. Having white and black is boring! We need to add two things here. First, in the body tag, add

<body style="background-color:blue;">

to your code. This will change the color of your background!
