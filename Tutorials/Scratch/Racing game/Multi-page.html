Scratch Racing Game

Finished files on GitHub

This is a basic tutorial that you should complete pretty quickly. You need to know how to use the scratch paint editor. This will lead you to make a racing game.

Ok! Let's get started. First off, delete the cat sprite and make a new one, name it Car1. Paint in a red racing car.

Let's code the car. Open the code window, and pull out the yellow block that says 'When space pressed'. Under it snap the turn [left arrow] 15 degrees. click the drop-down on the yellow block and select left arrow. Repeat for right turn. Now, if you press the green flag, the car should turn.

Page Three

But that's boring. Let's go forward! Drag out the yellow block saying 'When green flag clicked' and under it snap on the forever block. Inside the forever block, drag an if/else block. We are going to make the car move faster on the track. From the sensing drag the diamond-shaped 'if touching colour' block into the diamond-shaped hole in the if/else block. Click on the colour and click the eyedropper tool. Then click the grass. Inside the if part, drag out the 'move 10 steps' block, but change it to 2. Drag the same block into the else part, but make the value 7. If you click the green flag, the car goes forward!



Right-click on the car and select duplicate. In the new car, paint it blue, and then change the left and right arrow drop-downs to a and d. Now two people can race!



And you're done! Try playing around with the values to see what happens. Congratulations!
