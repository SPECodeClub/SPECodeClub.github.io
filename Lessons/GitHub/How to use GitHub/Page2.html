It's quite simple to use GitHub to download the files we need. First, click on the link. When you are on GitHub, you will see a button labeled 'Clone or download'.



Press this and select 'Download zip'.



Select the arrow on the popup at the bottom and select 'Show in folder'.



Right click on the file and select 'Extract all'.



Click 'Extract' to extract the files in the folder that you're in, or change the folder by pressing 'Browse'.



That's it! Inside the new folder will be your extracted files.
