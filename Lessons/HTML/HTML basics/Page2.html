HTML Documents
All HTML documents must start with a document type declaration: <!DOCTYPE html>.

The HTML document itself begins with <html> and ends with </html>.

The part of the HTML document that contains data about the page is between <head> and </head>.

The visible part of the HTML document is between <body> and </body>.

Some useful tags include:

- <p>[text goes here]<p> = a paragraph
- <h1>[heading goes here]</h1> = a heading. The lower the number (between 1 and 6), the larger the heading
- <img src="[image location]" alt="[alternative text - used if the image can't be shown or if narrator is on]" width="[width in pixels]" height="[height in pixels]"> = an image
- <a href="[link goes here]"> Link text</a> = a link
